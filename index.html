This project significantly improved campus sustainability by reducing carbon emissions from commuting by 30%. Features include user registration, ride scheduling, accepting rides, ride history, cancellation, and secure JWT-based session management.

Developed a dynamic web application to help people test for colorblindness. Built using HTML, CSS, JavaScript, Bootstrap, and Firebase. Achieved a 30% reduction in data processing time and a 20% improvement in system performance. The website includes 8 functional features and 4 non-functional features, enhancing efficiency by 61%.

A C++ program that finds the shortest path between airports, including paths with up to three layovers. Utilizes algorithms like Minimum Spanning Tree, BFS, and DFS. Also capable of finding round trips covering all adjacent airports. Contributed to enhancing route planning efficiency.
